Bluetooth Bike Shifter

Revolutionizing cycling with wireless shifting technology, enhancing performance and rider experience.

User Needs Features

Reliable

If conncetion dropped, automatically reconnects

Responsive

<0.5s from input to shift

Universal

Works with all common bicycle drivetrains

Development Process

The journey from concept to finished product involved extensive prototyping, rigorous testing, and continuous refinement. We faced challenges in miniaturizing the components and ensuring reliable performance across various riding conditions.

Future Improvements

We're exploring integration with cycling computers for data analysis, further reducing power consumption, and developing a pro version with advanced features for competitive cyclists.
